# Changelog

All notable changes to this project will be documented in this file.

## 0.0.2 - 2025-12-20

### Features

- **ftoc init**: Automatically download `google-patent-cli` and `arxiv-cli` binaries.
- **ftoc init**: Add `--insecure` flag to skip SSL verification during downloads.
- **CLI**: Remove legacy `toc` command and cleanup interface.

### Documentation

- Updated `README.md` to reflect new installation flow and remove outdated terminology.
- Updated prompt templates to reference local tool paths.
